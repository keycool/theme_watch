# 行业启动观察指标分层 V1

## 1. 定位

本文件只解决一个问题：

**哪些指标用于判断行业是否进入启动阶段，哪些指标只用于辅助确认、排序或风险提示。**

原则：

- 主指标决定主标签。
- 辅助指标不单独决定启动。
- 辅助指标可以降低优先级、提示过热或提示需要人工复核。
- 主指标偏中长期结构，辅助指标偏短期状态与风险。
- 主/辅分类不按一阶或二阶函数机械划分，而按指标在策略里的作用划分。

---

## 2. 主指标

主指标回答：

**这个行业是否真的具备“低位收敛后启动”的核心结构。**

### 2.1 结构层

用于判断行业是不是启动识别对象。

| 指标 | 用途 |
| --- | --- |
| 连续低于 MA250 的时间 | 判断是否经历过足够长的低位沉淀 |
| 偏离度σ：`(当前年线偏离 - 近120日年线偏离均值) / 近120日年线偏离标准差` | 判断当前位置相对年线偏离是否异常 |
| 低位区间 | 避免把高位回落误判成低位启动 |
| 收敛结构 | 判断是否出现整理、压缩、靠近年线的过程 |

说明：

- `年线偏离 = close / MA250 - 1`。
- 偏离度σ为正，表示收盘价在 MA250 上方。
- 偏离度σ为负，表示收盘价在 MA250 下方。
- 绝对值越大，表示当前偏离越明显。
- 偏离度本身不是启动确认，只是结构层证据。

### 2.2 突破层

用于判断行业是否从整理状态中走出。

| 指标 | 用途 |
| --- | --- |
| 是否近期站上 MA250 | 判断是否出现年线突破 |
| 是否带量突破 | 判断突破是否有资金配合 |
| 突破后是否站稳 | 排除单日脉冲 |

说明：

- 启动确认必须强调“近期突破”，不能只是长期站在年线上方。
- 已经在线上运行很久的行业，应优先归入趋势延续体系。

### 2.3 龙头层

用于判断板块内部是否有可代表行业的强势龙头。

| 指标 | 用途 |
| --- | --- |
| 龙头当日涨幅 >= 7% | 捕捉标志性大阳线 |
| 龙头 5 日强势 | 捕捉连续强于板块的龙头 |
| 龙头涨停 | 强确认，但不是唯一条件 |
| 龙头持续性 | 判断是否不是一日脉冲 |

说明：

- 龙头涨停有效，但不应成为唯一硬门槛。
- 连续强势有时比单日涨停更能代表行业启动质量。

## 3. 辅助指标

辅助指标回答：

**这个候选是否过热、是否拥挤、是否更值得优先看，或者是否需要人工复核。**

### 3.1 拥挤度与过热层

| 指标 | 用途 |
| --- | --- |
| 吸筹率分位 | 辅助识别是否已经过热或退潮 |
| 拥挤度标签 | 给主标签增加风险提示 |
| 过热退潮 | 排除“强势后消退”被误判为启动 |

说明：

- 拥挤度不替代主策略。
- `观察中 / 拥挤偏高` 的含义是：结构可能值得看，但风险不低。
- `趋势延续 / 过热预警` 的优先级应低于真正的低位启动候选。

### 3.2 波动率层

指数现货波动率建议先作为辅助指标：

```text
日内波动率 = (high - low) / close
波动率σ = (当前日内波动率 - 近120日日内波动率均值) / 近120日日内波动率标准差
```

用途：

- `波动率σ < 0`：当前波动低于自身历史常态，辅助观察低位压缩。
- `波动率σ > 0`：当前波动高于自身历史常态，辅助观察突破扩张或高位分歧。
- 绝对值越大，表示当前波动相对自身历史越异常。

当前不建议：

- 不用波动率单独判断启动。
- 不用波动率替代年线、低位收敛和龙头确认。
- 不因为波动率下降就直接给 `观察中` 或 `启动确认`。

### 3.3 映射质量层

| 指标 | 用途 |
| --- | --- |
| ETF 与申万二级相关性 | 判断对照关系是否可靠 |
| 成分行业重叠度 | 判断多个 ETF 是否适合合并观察 |
| 数据口径校验 | 排除价格复权、异常跳变造成的误判 |

说明：

- 映射质量只决定“看哪个申万二级”，不直接决定启动状态。
- 低相关时应先进入校验页，而不是正式主题页。

## 4. 决策口径

主标签只由主指标链条决定：

- `未启动`
- `观察中`
- `接近启动`
- `启动确认`
- `趋势延续型偏强/强势`

辅助标签只做叠加：

- `拥挤正常`
- `拥挤偏高`
- `过热预警`
- `过热退潮`
- `波动压缩`
- `波动扩张`
- `映射待校验`

推荐展示格式：

```text
主标签 / 辅助标签
```

示例：

- `观察中 / 拥挤正常`
- `未启动 / 波动压缩`
- `趋势延续 / 过热预警`
- `启动确认 / 拥挤偏高`

## 5. 评分制度适配

可以借鉴均值回归模型的三段式思想，但不直接照搬配置模型的权重。

原配置模型的思想是：

- 分位看长期位置。
- 趋势看最近方向。
- 均值回归看短期是否拉得太过。

在行业启动策略里，适配为：

- 中长期结构评分：对应低位、年线、收敛和突破前结构。
- 趋势确认评分：对应近期突破、站稳和龙头转强。
- 风险修正评分：对应吸筹率分位、拥挤度、波动率σ和过热退潮。

当前原则：

- `启动阶段` 由中长期结构和趋势确认决定。
- `风险修正` 不单独决定启动，只影响观察优先级和风险提示。
- 不把所有指标合成一个单一总分，避免辅助指标把过热行业误推成启动确认。

推荐理解：

```text
启动阶段 = 主评分决定
观察优先级 = 主评分 + 辅助风险修正
```

## 6. 当前实施建议

短期先做：

- 页面展示上区分主指标和辅助指标。
- 把波动率放入辅助指标区。
- 偏离度σ采用均值回归 zscore 口径：先计算年线偏离，再看当前偏离相对近120日均值偏离几个标准差。
- 主标签仍按年线结构、突破行为、龙头确认来判断。

暂不做：

- 不把波动率并入启动确认公式。
- 不用辅助指标重写现有主策略。
- 不为了新增指标扩大解释文字，先保持页面清爽。
